### Project Title: Stock Market Price Prediction System using Deep Learning
**Technologies:** Python, TensorFlow/Keras, Streamlit, Pandas, Scikit-learn, yFinance

*   Designed and developed an end-to-end Deep Learning system to forecast financial market trends using Long Short-Term Memory (LSTM) neural networks.
*   Automated historical data pipeline by integrating the `yfinance` API to extract and preprocess large time-series datasets (Open, High, Low, Close, Volume) dynamically.
*   Engineered data preprocessing modules utilizing `MinMaxScaler` and sliding-window sequence generation to optimize the neural network's convergence and accuracy.
*   Architected a multi-layered LSTM model featuring Dropout regularization (20%) to mitigate overfitting, achieving robust performance measured by low RMSE and MAE scores.
*   Built and deployed an interactive web application using Streamlit and Plotly, enabling users to input real-time stock tickers, visualize Exploratory Data Analysis (EDA), and view "Next Day" predictive modeling insights.
